# Critical Node Detection using CRITIC-TOPSIS Framework

## Vision
A comprehensive framework and interactive pipeline for identifying critical nodes in complex networks. It combines multiple centrality measures using an objective weighting mechanism (CRITIC) and a similarity-based ranking system (TOPSIS).

## Architecture
- **Core Algorithms:** Centrality calculation, CRITIC weighting, TOPSIS ranking.
- **Evaluation Engine:** Targeted attack simulations, cascading failures, robustness testing.
- **Advanced Modules:** Adversarial testing, temporal evolution, domain-specific intelligence, uncertainty quantification, explainable AI.
- **Frontend:** Interactive 6-tab Streamlit dashboard (`app.py`).

## Tech Stack
- Python 3.8+
- NetworkX, NumPy, Pandas, SciPy, Scikit-Learn
- Plotly, Streamlit, Matplotlib, Seaborn
