# Requirements & Scope

## Existing Features
- [x] Compute 7 core centrality measures.
- [x] CRITIC method for variance-based, objective weighting.
- [x] TOPSIS method for multi-attribute ranking.
- [x] Targeted Attack Simulation & Cascading failures.
- [x] Explainable AI (XAI) feature importance for node criticality.
- [x] Streamlit App with Visualization & Interactive Pipeline.
- [x] Sensitivity, Uncertainty, and Adversarial analysis.
- [x] Temporal tracking of node rankings.

## Missing Essentials (v1 Scope & Beyond)
- [ ] Complete API abstraction (FastAPI) to decouple UI from algorithms.
- [ ] Scalability optimizations for huge graphs (e.g. cuGraph/GPU acceleration).
- [ ] Database persistence for saving network runs, temporal histories and UI states.
- [ ] Modularization of the frontend `app.py` for maintainability.
